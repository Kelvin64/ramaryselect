# Ramary Select Wine Website

A modern, responsive website for showcasing and selling premium wines.

## Project Structure

```
ramaryselect/
├── assets/          # Static assets like images, fonts, and icons
├── css/            # Stylesheets
│   ├── components/ # Component-specific styles
│   └── utils/      # Utility styles and variables
├── js/             # JavaScript files
└── *.html          # HTML pages
```

## Features

- Responsive design
- Modern UI/UX
- Wine showcase
- About section
- Interactive elements

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start developing!

## Development

- Vanilla JavaScript
